# 赛鸽血统环号登记站

## 快速开始

### 运行方式

```bash
npm start
```

启动后访问 `http://localhost:3024`。支持档案管理、血统查询、转让审核、疫苗接种、配对计划和归巢成绩记录。

## 批量导入功能

### 功能入口

点击页面顶部导航栏的 **「批量导入」** 按钮，打开批量导入弹窗。

### 操作步骤

1. **下载CSV模板**：点击弹窗中的「↓ 下载CSV模板」按钮，下载包含标准表头的CSV模板文件
2. **编辑数据**：在Excel、WPS或其他表格软件中打开模板，按照下方字段说明填写数据
3. **粘贴导入**：将编辑好的CSV内容（包括表头行）复制粘贴到文本框中
4. **预览验证**：点击「预览解析结果」查看数据有效性，系统会自动标记必填项缺失、足环号重复等错误
5. **确认导入**：核对无误后点击「确认导入」完成批量录入

### 快捷操作

- **填入示例数据并预览**：点击「✎ 填入示例数据并预览」按钮，自动填充6条示例鸽只数据并展示解析结果，便于快速了解导入格式和预览效果

### CSV模板字段说明

| 字段名称（中文表头） | 英文/别名 | 是否必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **足环号** | ringNo、环号 | ✅ 必填 | 鸽只唯一标识，不可与现有档案重复，格式如 CHN-2026-001 |
| **鸽主** | owner | ✅ 必填 | 当前归属鸽舍或鸽主名称，如：北岸棚、育种棚 |
| **父环号** | fatherRing、父鸽环号、父亲环号 | ❌ 选填 | 父鸽足环号，如父鸽未在系统中登记也可填写，留空表示未知 |
| **母环号** | motherRing、母鸽环号、母亲环号 | ❌ 选填 | 母鸽足环号，如母鸽未在系统中登记也可填写，留空表示未知 |
| **羽色** | color | ✅ 必填 | 羽毛颜色，如：灰、雨点、红轮、绛、白花、石板、黑等 |
| **棚号** | loft、出生棚、出生棚号 | ✅ 必填 | 出生或所属鸽棚名称，如：北岸A棚、种鸽棚、南岸鸽棚 |

### 表头兼容说明

系统支持多种表头写法（不区分大小写，列顺序任意），以下写法均会被正确识别：

- 足环号 / 环号 / ringNo / ring_no / ring-no
- 鸽主 / owner
- 父环号 / 父鸽环号 / 父亲环号 / fatherRing / father_ring / father-ring
- 母环号 / 母鸽环号 / 母亲环号 / motherRing / mother_ring / mother-ring
- 羽色 / color
- 棚号 / 出生棚 / 出生棚号 / loft

### 示例CSV内容

```csv
足环号,鸽主,父环号,母环号,羽色,棚号
CHN-2026-100,北岸棚,CHN-2022-188,CHN-2023-512,灰,北岸A棚
CHN-2026-101,北岸棚,,,雨点,北岸B棚
CHN-2026-102,南岸棚,,,绛,南岸鸽棚
CHN-2026-103,北岸棚,CHN-2022-188,,白花,北岸A棚
CHN-2026-104,育种棚,,CHN-2023-512,红轮,种鸽棚
CHN-2026-105,育种棚,CHN-2022-188,CHN-2023-512,石板,种鸽棚
```

示例数据包含以下典型场景：
- **CHN-2026-100**：父母信息完整
- **CHN-2026-101 / CHN-2026-102**：父母信息均未知
- **CHN-2026-103**：仅登记父环号
- **CHN-2026-104**：仅登记母环号
- **CHN-2026-105**：父母信息完整（与100为同父母兄弟姐妹）

### 导入校验规则

- **必填校验**：足环号、鸽主、羽色、棚号四项为必填，缺失将标记为无效记录
- **足环号唯一性**：
  - 不得与系统中已存在的足环号重复
  - 同一批次导入数据内不得出现重复足环号
- **有效记录**：校验通过的记录显示为「✓ 有效」，可参与导入
- **无效记录**：校验不通过的记录以红色高亮，并显示具体错误标签，导入时将被自动跳过

### 运行验证方式

启动服务后，可按以下步骤验证批量导入功能：

1. **启动服务**：执行 `npm start`，浏览器访问 `http://localhost:3024`
2. **打开导入弹窗**：点击顶部「批量导入」按钮
3. **验证模板下载**：
   - 点击「↓ 下载CSV模板」
   - 确认自动下载 `pigeon-import-template-YYYY-MM-DD.csv` 文件
   - 打开文件确认表头为：`足环号,鸽主,父环号,母环号,羽色,棚号`
4. **验证示例数据**：
   - 点击「✎ 填入示例数据并预览」
   - 确认文本框自动填入6条示例数据
   - 确认预览表格显示统计信息和每行数据校验状态
   - 确认其中包含：完整父母信息、仅父环号、仅母环号、无父母、部分字段为空等多种场景
5. **验证自定义导入**：
   - 清空文本框，手动输入或粘贴一段CSV数据
   - 点击「预览解析结果」，确认校验逻辑正常（必填缺失会显示红色错误标签）
   - 点击「确认导入」，确认有效记录被录入，首页卡片列表新增对应鸽只
6. **API验证（可选）**：

   ```bash
   # 下载模板API
   curl -O http://localhost:3024/api/pigeons/import/template
   
   # 获取示例数据API
   curl http://localhost:3024/api/pigeons/import/sample
   
   # 预览解析API
   curl -X POST http://localhost:3024/api/pigeons/import/preview \
     -H "Content-Type: application/json" \
     -d '{"csv":"足环号,鸽主,父环号,母环号,羽色,棚号\nCHN-TEST-001,测试棚,,,灰,测试棚"}'
   ```
